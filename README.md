## Описание

HelpDesk — система управления заявками. 

## Функционал

- Просмотр списка тикетов
- Создание нового тикета
- Редактирование тикета
- Удаление тикета
- Отметка о выполнении
- Просмотр детального описания

## Технологии

- Webpack
- JavaScript
- GitHub Actions
- GitHub Pages
